# 2026-04-16

## PRD 版本提醒机制

- 为 `PawMind-PRD-v2.0.html` 和 `PawMind-PRD-v2.1.html` 添加了**动态版本探测 banner**。
- 逻辑：每个 PRD 声明自身版本号（`PRD_VERSION = [major, minor]`），页面加载时通过 `fetch HEAD` 逐版探测同目录下是否存在更新版本文件，找到则显示升级 banner，点击跳转，右侧 ✕ 可关闭。
- 无更新版本时静默不显示。
- 后续新增 v2.2 只需放文件到同目录，v2.1 会自动检测并提示，无需手动改任何代码。
