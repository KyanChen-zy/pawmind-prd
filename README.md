# PawMind — AI 宠物健康管理平台

> 让 AI 成为宠物的私人健康管家

PawMind 是面向**城市养宠人群**的 AI 宠物健康管理 App，通过智能设备数据采集、健康指标可视化、AI 辅助诊断和健康知识服务，帮助主人全面掌握宠物健康状况，及时发现健康隐患。

```
多渠道健康数据（设备自动上报 / 手动录入 / 拍照扫描）
  → 数据聚合与指标计算
  → 健康指标可视化（24h / 7天）
  → AI 健康分析（异常检测 / 诊断 / 问答）
  → 主人知情 & 行动（预警 / 报告 / 就医建议）
```

## 核心功能

### 🩺 健康管理

- **健康指标卡** — 24h / 7 天核心健康指标汇总，一眼掌握宠物状态
- **数据可视化** — 各指标趋势折线图，支持时间范围切换
- **设备驱动显示** — 已绑定设备展示实时数据，未绑定显示营销引导
- **手动录入** — 体重、饮食量、饮水量、体温、排便情况手动记录
- **异常预警** — 基于指标阈值检测异常，分级推送通知并给出行动建议

### 📱 设备管理

- **设备列表** — 已绑定设备卡片展示（名称、图标、电量、连接状态）
- **配对绑定** — 选择设备类型 → 配对引导 → 绑定确认，支持主流品牌
- **设备详情** — 名称编辑、电量/网络状态、产品信息查看
- **设备商城** — 展示型商城，设备参数介绍 + 外部购买链接导流

### 📋 健康档案

- **就诊记录** — 诊断结果、处方用药、医嘱、附件图片完整记录
- **日常观察** — 自由文本 + 标签，灵活记录日常健康情况
- **多输入方式** — 键盘打字 / OCR 拍照识别 / 语音输入，3 秒完成记录
- **档案时间线** — 按时间倒序展示，支持编辑和删除

### 💉 电子疫苗本

- **条形码扫描** — 相机扫描疫苗条形码，OCR 识别并预填疫苗名称
- **疫苗管理** — 已接种记录 + 待接种提醒，过期未接种红色高亮
- **接种提醒** — 基于下次接种日期推送本地通知

### 🤖 CareAI 智能助手

- **健康知识问答** — 对话式交互，AI 结合宠物档案上下文回答健康问题
- **AI 拍照诊断** — 口腔检查 / 粪便检查 / 皮肤检查 / 报告解读 / 药品识别
- **每日小知识** — 按分类组织的宠物健康知识推送，支持历史回看
- **免责声明** — 所有 AI 分析结果附带「仅供参考，不替代专业诊断」声明

### 📸 成长记录

- **照片/视频上传** — 随时记录宠物精彩瞬间
- **时间轴展示** — 按时间倒序浏览所有成长内容
- **里程碑标记** — 标注首次疫苗、生日等特殊事件
- **AI 月度报告** — 每月自动生成图文成长总结，支持一键分享

## 技术架构

| 层级 | 技术选型 |
|------|----------|
| 前端框架 | React Native + Expo |
| 状态管理 | Zustand |
| 后端框架 | NestJS + TypeORM |
| 数据库 | PostgreSQL |
| AI 能力 | 大语言模型 API（GPT-4o / 混元），Prompt 注入宠物档案上下文 |
| 异常检测 | 规则引擎（阈值检测），后期引入时序异常检测模型 |
| 推送通知 | iOS APNs + Android FCM |
| 图片存储 | CDN + OSS，客户端压缩至 1MB 以内 |
| 设备联动 | MQTT 协议，兼容主流品牌（小米 / 希喂 / 猫猫机等） |
| 隐私合规 | 健康数据本地加密存储，遵循 GDPR / 个保法 |

## 文档说明

### PRD 产品需求文档

`prd/` 目录包含 PawMind v2.0 完整产品需求文档，涵盖：

- 产品架构总览与信息层级
- 核心数据模型设计
- 用户画像与用户旅程
- 六大功能模块详细方案（健康管理、设备管理、健康档案、电子疫苗本、CareAI、成长记录）
- 异常与边界处理方案
- AI 能力定位与设计原则
- 竞品对比与差异化分析

> 推荐使用浏览器打开 `.html` 版本获得更好的阅读体验。

### 业务流程图

`flowcharts/` 目录包含 7 个核心业务流程的 Mermaid 格式流程图：

| 文件 | 描述 |
|------|------|
| `health-data-flow.mmd` | 健康指标卡加载、数据可视化、设备驱动显示逻辑 |
| `device-bind-flow.mmd` | 设备添加、配对引导、绑定确认、设备管理全流程 |
| `health-record-flow.mmd` | 就诊记录/日常观察录入，OCR、语音等多输入方式 |
| `vaccine-scan-flow.mmd` | 疫苗条形码扫描、OCR 预填、接种提醒注册流程 |
| `careai-chat-flow.mmd` | 健康问答会话管理、消息发送与 AI 回复流程 |
| `ai-diagnosis-flow.mmd` | AI 拍照诊断统一流程（口腔/粪便/皮肤/报告/药品） |
| `alert-flow.mmd` | 健康异常检测 → 预警推送 → 主人处理全链路时序图 |

> `.mmd` 文件可使用 [Mermaid Live Editor](https://mermaid.live/) 或支持 Mermaid 的编辑器（如 VS Code + Mermaid 插件）查看。

### 交互原型

`prototype/` 目录包含可交互的 HTML 原型，浏览器直接打开即可体验主要页面和交互流程。
