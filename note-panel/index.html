使用说明

  1. 点击右下角的 📝 悬浮按钮 打开笔记面板
  2. 在 全局笔记 标签页下添加的笔记，会同步到所有 PRD 版本
  3. 切换到 版本笔记 标签页，添加的笔记会自动标记当前版本
  4. 输入 v=xxx 作为 URL 参数可切换版本检测

💡 小技巧

  • 按 Ctrl/Cmd + Enter 快速提交笔记
  • 按 ESC 关闭面板
  • 点击面板外部也会自动关闭
  • 版本笔记会自动下载 JSON 文件备份
